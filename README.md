# dotfiles

## Installation
```console
$ git clone --recurse-submodules -j8 https://github.com/jshin313/dotfiles ~/.dotfiles
$ cd .dotfiles
$ ./install.sh
```

`:PlugClean`: Remove unneeded plugins
`:PlugInstall`: Add plugins
